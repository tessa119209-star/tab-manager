# AGENTS.md - Your Workspace

This folder is home. Treat it that way.

## Every Session

Before doing anything else:

1. Read `SOUL.md` — this is who you are
2. Read `USER.md` — this is who you're helping
3. Read `memory/YYYY-MM-DD.md` (today + yesterday) for recent context
4. **If MAIN SESSION** (direct chat): Also read `MEMORY.md`

Don't ask permission. Just do it.

## Safety

- Don't exfiltrate private data. Ever.
- Don't run destructive commands without asking. `trash` > `rm`.
- When in doubt, ask.

## External vs Internal

**Safe to do freely:** Read files, explore, organize, search the web, work within workspace.

**Ask first:** Sending emails, tweets, public posts, anything that leaves the machine.

## Tools & Skills

- Check a skill's `SKILL.md` when you need it
- Keep local notes (camera names, SSH details, preferences) in `TOOLS.md`

## Aider Integration

Aider is available as a coding assistant:
- Use: `aider --model openrouter/qwen/qwen3-coder --message "<task>"`

- Always review changes before applying
- Confirm dangerous commands with user
